# TextAliveAppAPI-base
textalive-app-api(https://github.com/TextAliveJp/textalive-app-api) を使って(厳密にはインストール不要)最小限のコードで以下の動作ができるようにしました。<br>
・曲名とアーティスト名の表示<br>
・音楽の再生と停止<br>
・歌詞の文字単位での表示<br>
<h2>実行方法</h2>
Node.jsがインストールされた環境でコマンドプロンプトに<br>
<strong>npm run dev</strong><br>
と打ち込むと実行されます。<br>
